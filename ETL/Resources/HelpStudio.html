ETL Studio Help

What is this application?
ETL Studio allows you to migrate data between several databases.
You should create a Job where you list the migrations (called ETL) that you wish to do in this job, in a specific order. Then you run this job and the data will be migrated.

Why ETL Studio?
Our application supports several database types:
  • MySQL
  • MS Access
  • PostgreSQL
  • SQL Server
  • ODBC
We also support migration to DSpace library management.

How to use ETL Studio?
There are several easy steps to go through:
  • Connect to the databases (source and destination)
  • Create views (called Query) that is a JOIN query by right clicking on the 'Queries' node in the database in the treeview on the right. (Optional step)
  • Create an ETL by choosing source database and table (or query that was created in step 2), and destination database and table, as well as the transformation
  • Create a Job and add the ETLs needed to it
  • Run the Job
You can find more help pages for each step in the correspondent window.

Enjoy!
