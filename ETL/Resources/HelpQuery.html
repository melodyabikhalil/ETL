Query Help

This page consists of creating a new query (view) for a database.
First you should choose a name for your new query.
Then a main table which is the one the query will start with.
For example: SELECT * FROM x JOIN y ON x.id=y.id. Here, x is the main table.

Table columns usage:
Example: SELECT * FROM users INNER JOIN
  • Join type:
    • Type of the join query: INNER, OUTER, LEFT, RIGHT, FULL.

  • Table 1:
    • Table name after the join

  • Column 1:
    • Column to join on from table 1

  • Table 2:
    • Table name to join columns with

  • Column 2:
    • Column to join on from table 2


Example:
Query:
SELECT * FROM Users
INNER JOIN Languages ON Languages.ID=Users.language_id
LEFT JOIN Addresses ON Addresses.ID=Users.address_id
RIGHT JOIN Buildings ON Buildings.ID=Addresses.building_id

Main table: Users
Table to fill:
Join Type Table 1 Column 1 Table 2 Column 2
INNER JOIN Languages ID Users language_id
LEFT JOIN Addresses ID Users address_id
RIGHT JOIN Buildings ID Addresses building_id
