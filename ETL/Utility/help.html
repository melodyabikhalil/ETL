There is three types of expressions: Replace, Reg(regular expression) and Map.

Usage examples:

REPLACE: table1, FullName, Replace, null, [FirstName].[LastName], null REGULAR EXPRESSION: table1, Gender, Reg, Gender, ^[A-Za-z]{2}, null (takes the first two letters of the gender value) MAPPING: table1, Gender, Map, Gender, null, Gender (maps the gender value to another one using the mapping table)
